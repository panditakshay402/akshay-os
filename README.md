-  Three.js 3D Elements
-  Smooth Animations
-  Fast Performance
-  Responsive Design
-  Modern Portfolio Layout

# ⚙️ React + Vite Template

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses **Oxc**
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses **SWC**

---

#  React Compiler

The React Compiler is **not enabled** on this template because of its impact on development and build performance.

To enable it, refer to the official React documentation.

---

#  ESLint Configuration

If you are developing a production-ready application, it is recommended to use **TypeScript** with type-aware lint rules enabled.

You can check the official **React + TypeScript Template** for more information.

#  Project Setup

## Create Vite Project

```bash
npm create vite@latest .
```

---

#  Install Main Packages

```bash
npm install three @react-three/fiber @react-three/drei framer-motion gsap react-icons react-router-dom lenis
```

---

# 📚 Package Explanation

| Package | Purpose |
|----------|---------|
| 🎮 three | 3D Graphics Library |
| ⚛️ @react-three/fiber | Use Three.js with React |
| 🧩 @react-three/drei | Ready-made helpers for Three.js |
| 🎬 framer-motion | UI Animations |
| 🚀 gsap | Advanced Scroll & Timeline Animations |
| 🎯 react-icons | Icon Library |
| 🛣 react-router-dom | Routing (Future Use) |
| 🌊 lenis | Smooth Scrolling |
